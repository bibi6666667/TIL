### 1. 논리적 추론

1거짓 2진실

i) A가 거짓이라면 : A 안먹, A안먹, B먹 (모순x)

ii) B가 거짓이라면 : A참, A먹, B먹 (모순)

iii) C가 거짓이라면 : B안먹, A먹, A안먹 (모순)

따라서 거짓말을 한 사람은 A, 빵을 먹은 사람은 B

### 5. 논리식

윤년인 경우

year %4 == 0 이고 year %100 != 0 

year %4 == 0 이고 year %400 == 0

-> year %4 == 0 이고 (year %100 != 0 이거나 year %400 == 0)

따라서 답은 2번
