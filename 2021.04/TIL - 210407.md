#### card 내의 내용

- Long id
- String title
- Date date
- String contents 
- boolean todo
- boolean doing
- boolean done



## URL convention with 쏭

- crate
  - POST "/cards"
- read
  - GET "/cards"
- move
  - PUT "/cards/{id}/todo"
  - PUT "/cards/{id}/doing"
  - PUT "/cards/{id}/done"
- update
  - PUT "/cards/{id}"
- delete
  - DELETE "/cards/{id}"
